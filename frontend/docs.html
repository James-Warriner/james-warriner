API Documentation

Retrieve data about me:

  • /api/about_me?section={insert}

Available sections:

  • Name
  • Location
  • Github
  • Experience
  • Personal Projects

Example Request:

curl -X GET https://api/about_me?section=Name

Example Response:

{
  "Name": "James Warriner"
}

Fetch a single random fact, weighted by rarity.

Example Request:

curl -X GET https://yourdomain.com/api/fact

Example Response:

{
  "fact": "I have a fox red labrador named Bruno",
  "rarity": "Uncommon"
}
